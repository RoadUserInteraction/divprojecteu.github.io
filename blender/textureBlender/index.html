Add texture

March 28, 2017
  1. Create a plane (Shift + A)
  2. Add a material by pressing new in the material tab.
  3. Add a texture by pressing new in the texture tab.
  4. Open the picture you want as texture.
  5. Change the view to UV-editing.
  6. Choose your picture in the list.
  7. Press TAB to go into edit mode and choose Unwrap (U) to paste the picture onto the plane. Make sure you are in Texture display mode
  8. Duplicate object (Shift + D) and snap the object with CTRL.
  9. Move selected object with G.
